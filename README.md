## Booking Engine ##

![Scala](https://img.shields.io/badge/scala-%23DC322F.svg?style=for-the-badge&logo=scala&logoColor=white)
 
A simple booking engine using Scala, Akka Http and database PostgresSQL with migration using flyway
## Built with ##

- Scala 3.3.1 (Java 21)
- Postgres 16.2
- Sbt 1.9.7
